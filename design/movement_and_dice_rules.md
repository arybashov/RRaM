# Правила движения и кубиков

## 1. Борды

Борды - это черная ромбообразная сетка на карте. Перемещение персонажей происходит по этим ячейкам.

## 2. Броски

В начале игры каждому игроку доступно 10 бросков.

Один бросок состоит из двух кубиков.

## 3. Использование двух кубиков

Игрок выбирает, как использовать значения кубиков:

### Вариант 1: общее движение

Игрок складывает значения двух кубиков и перемещает выбранного персонажа на получившееся количество бордов.

Пример:

- Первый кубик: `3`.
- Второй кубик: `4`.
- Персонаж может пройти `7` бордов.

### Вариант 2: разделение действий

Игрок использует кубики отдельно:

- Один кубик - для передачи карт между персонажами.
- Второй кубик - для добора карты или движения.

## 4. Передача карт

Значение кубика при передаче карт задает лимит передаваемых карт.

Пример:

- Кубик: `2`.
- Можно передать до `2` карт от одного персонажа другому.

Требует уточнения:

- Должны ли персонажи стоять на одной борде?
- Можно ли передавать карты на расстоянии?
- Можно ли передавать карты между персонажами разных игроков?
- Можно ли передавать любые карты или только предметы?

## 5. Добор карты

Значение кубика при доборе карты не влияет на количество карт.

Пример:

- Кубик: `5`.
- Игрок берет `1` карту.

Требует уточнения:

- Из какой колоды можно брать карту?
- Можно ли выбирать колоду свободно?
- Есть ли цена или условие для добора?
- Можно ли добирать карту любым персонажем?

## 6. Движение одним кубиком

Если один кубик используется для движения, персонаж проходит количество бордов, равное значению этого кубика.

Пример:

- Кубик: `5`.
- Персонаж может пройти `5` бордов.
