# RRaM - утвержденные решения

Этот файл нужен, чтобы фиксировать решения после обсуждения и не возвращаться к одним и тем же вопросам.

## Уже принято

- Проект называется `RRaM`.
- Формат: онлайн RTS/RPG.
- Игроки начинают на противоположных островах.
- На карте есть ромбообразная сетка бордов.
- Основные стартовые персонажи:
  - `К` - кузнец.
  - `П` - помощник кузнеца.
  - `В` - воин.
  - `О` - охотник.
  - `Ш` - шаман.
- Подписанные стартовые точки используются для начальной расстановки.
- Эти же точки могут быть целями телепортации через карту `Бусы телепортации`.
- Каждому игроку изначально дается 10 бросков.
- Один бросок включает два кубика.
- Два кубика можно сложить для движения.
- Один кубик можно потратить на передачу карт.
- Второй кубик можно потратить на добор карты или движение.
- При передаче карт значение кубика задает лимит карт.
- При доборе карты значение кубика не увеличивает количество карт: берется одна карта.

## Нужно подтвердить

- Значение `10 бросков`.
- Правила боя.
- Правила победы.
- Ограничения на передачу карт.
- Ограничения на телепортацию.
- Размер и устройство карты.
- Состав колод.
