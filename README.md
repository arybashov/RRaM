# RRaM

Онлайн RTS/RPG проект с островной картой, персонажами, карточной системой и перемещением по бордам через кубики.

## Структура

- `design/` - геймдизайн, правила, персонажи, карты, баланс.
- `tech/` - техническое ТЗ, архитектура, сущности, сетевые требования.
- `assets/` - будущие изображения, схемы карт, референсы.
- `notes/` - черновики, идеи, неструктурированные материалы.

## Текущий статус

Создан первый черновик правил по стартовым позициям, бордам, кубикам, передаче карт, добору карт и телепортации.
