# RRaM

Онлайн RTS/RPG проект с островной картой, персонажами, карточной системой и перемещением по бордам через кубики.

## Структура

- `design/` - геймдизайн, правила, персонажи, карты, баланс.
- `tech/` - техническое ТЗ, архитектура, сущности, сетевые требования.
- `assets/` - будущие изображения, схемы карт, референсы.
- `notes/` - черновики, идеи, неструктурированные материалы.

## Текущий статус

Цель текущего этапа: играбельный онлайн-прототип `RTS/RPG and Maps (RRaM)`.

Собран первый веб-прототип с картой, кубиками, ходами, персонажами, инвентарем, передачей/добором карт, телепортацией и журналом событий.

Демо: https://arybashov.github.io/RRaM/

Рабочий статус зафиксирован в `notes/status_report_2026-06-05.md`.
