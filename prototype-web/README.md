# RRaM web prototype

Первый локальный браузерный прототип.

## Запуск

Можно открыть файл напрямую:

```text
/home/aryba/Документы/RRaM/prototype-web/index.html
```

Или запустить локальный статический сервер из этой папки:

```bash
python3 -m http.server 8080
```

После этого открыть:

```text
http://127.0.0.1:8080
```

## Что уже есть

- Два игрока.
- Противоположные стартовые острова.
- Персонажи `К`, `П`, `В`, `О`, `Ш`.
- Ромбообразная сетка бордов.
- Бросок двух кубиков.
- Движение суммой двух кубиков.
- Движение одним выбранным кубиком.
- Добор одной карты.
- Передача карт по лимиту кубика.
- Карта `Бусы телепортации`.
- Журнал действий.

## Ограничения

Это не финальная онлайн-версия, а проверка базовых правил и темпа хода. Сетевой сервер, авторизация, сохранение матчей и честная серверная валидация будут вынесены в отдельный серверный прототип.
